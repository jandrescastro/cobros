# Cobros Mensuales App

Base inicial en `Next.js + TypeScript + Tailwind CSS + Supabase` para gestionar cobros mensuales desde móvil.

## Incluye

- Dashboard con resumen del mes
- Lista mensual con acción para marcar pagos
- Gestión base de clientes
- Pendientes acumulados e historial por cliente
- Mocks y utilidades listas para migrar a Supabase

## Pasos sugeridos

1. Instalar dependencias con `npm install`
2. Copiar `.env.example` a `.env.local`
3. Ejecutar `npm run dev`
